# url shortener

## To start:

> git clone https://github.com/akourtis98/url_shortener

> cd .\url_shortener\

> npm i

> use the "db structure.sql" to make the database

> create a .env file at the root of the project

`it should like this:`

```
DB_HOST=...
DB_USER=...
DB_PASS=...
DB_DATABASE=...
```

> npm run start

OR

> npm run dev
